# 代码美化器 - 项目计划

## 🎯 产品定位

**名称**: CodeBeauty - 一键美化网页代码块

**一句话**: 让AI输出的代码从"垃圾"变成"艺术品"

**核心价值**:
- 开发者每天看大量AI生成的代码，但格式混乱、难以阅读
- 一键转换：丑陋 → 美观、难读 → 易读

## 📊 市场验证

✅ **Hacker News验证**: "Claude Code CLI美化工具" 57票
✅ **真实痛点**: AI输出代码格式混乱
✅ **目标用户**: 开发者、技术学生、AI重度用户

## 🚀 MVP功能范围（3-5天完成）

### Phase 1: 核心功能（2天）
- [x] 检测网页代码块（`<pre>`, `<code>`）
- [x] 一键美化（语法高亮、行号）
- [x] 复制到剪贴板
- [x] 自适应主题（亮/暗）

### Phase 2: 增强功能（1-2天）
- [x] 导出PNG图片
- [x] 自定义主题选择
- [x] 代码折叠
- [x] 多语言支持

### Phase 3: 收费功能（第3天）
- [x] Pro主题库
- [x] 批量导出
- [x] 自定义字体
- [x] 去水印

## 💰 收费模式

**免费版**:
- 基础美化
- 3个默认主题
- 单个导出（带水印）

**Pro版 - $5/月**:
- 20+高级主题
- 批量导出
- 无水印
- 自定义配置
- 优先支持

## 🛠️ 技术栈

**前端**:
- Chrome Extension API
- Vanilla JS（轻量快速）
- Prism.js（语法高亮）
- html2canvas（导出图片）

**后端**:
- 无需后端（纯客户端）

**部署**:
- Chrome Web Store
- Edge Add-ons
- Firefox Add-ons
