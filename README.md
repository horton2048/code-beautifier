# CodeBeauty - 代码美化器

一键美化网页上的代码块，让代码更易读、更美观。

## ✨ 功能

- 🎨 一键美化所有代码块
- 🌈 多种主题（Monokai, GitHub, Dracula, Nord）
- 📏 可调字号（12-18px）
- 🔢 行号显示
- 📋 一键复制
- 📸 导出PNG图片

## 🚀 安装

### 开发模式

1. 克隆或下载此项目
2. 打开Chrome浏览器，访问 `chrome://extensions/`
3. 开启"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择项目文件夹

### 从Chrome商店安装

（即将上架）

## 📖 使用

1. 访问任何包含代码的网页
2. 点击浏览器工具栏中的CodeBeauty图标
3. 点击"美化所有代码"
4. 完成！

- Chrome Extension API
- Vanilla JavaScript
- Prism.js（语法高亮）

## 📝 开发

```bash
# 克隆项目
git clone https://github.com/yourusername/code-beautifier.git

# 进入项目目录
cd code-beautifier

# 在Chrome中加载扩展
chrome://extensions/ → 开发者模式 → 加载已解压的扩展
```
